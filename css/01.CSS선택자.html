CSS 선택자 1

1. 모든 선택자(*)

HTML 문서 안의 모든 태그를 선택할 때 전체 선택자를 사용한다. 
        *{
            스타일속성: 값;
            스타일속성: 값;
            ...
        }

2. 태그 선택자

태그명을 이용해 요소를 선택 
        태그명{
            스타일속성: 값;
            스타일속성: 값;
            ...
        }

3. 클래스 선택자(.class명)

HTML 문서 내에 여러 개의 태그를 동일한 클래스명으로 지정하여 선택할 때 사용 
        .class명{
            스타일속성: 값;
            스타일속성: 값;
            ...
        }

4. 아이디 선택자(#id명)

요소의 id속성을 이용하여 요소를 선택 
        #id명{
            스타일속성: 값;
            스타일속성: 값;
            ...
        }
