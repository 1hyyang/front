BOM(Browser Object Model)

window 객체

window 객체는 자바스크립트의 최상위 객체이며, 브라우저 내의 모든 요소들과 객체를 포함한다.

1) window.open()

2) 타이머 관련 메소드

특정 함수를 일정 시간이 경과된 이후에 호출할 수 있도록 제공되는 함수

window.setTimeout()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 한 번만 실행한다.

window.setInterval()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 반복해서 실행한다.

BOM(Browser Object Model) 객체

1) location 객체

브라우저의 주소 표시줄과 관련된 객체이다.

location.reload()

페이지를 새로고침하는 메소드

location.href

현재 페이지의 url 주소를 가지고 있는 속성으로 해당 속성의 값을 변경해서 페이지를 이동할 수 있다.

location.assign() : 페이지 이동

location.replace() : 페이지 이동, history에 기록되지 않는다.(뒤로가기 불가)

2) history 객체

브라우저의 방문 기록과 관련된 객체이다. 방문한 사이트의 주소가 배열 형태로 저장된다.

3) navigator 객체

웹 페이지를 실행하고 있는 브라우저에 대한 정보를 가지고 있는 객체이다.

4) screen 객체

운영체제의 화면(모니터)의 정보를 가지고 있는 객체이다.
